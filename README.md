
# PassTrip

Um site que tem como foco auxiliar os usuários identificarem as principais informações necessárias sobre vistos para onde desejam viajar. 



[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)



## Funcionalidades

- Visualizar informações sobre vistos
- Detalhes sobre o país de destino


## Stack utilizada

**Back-end:** .NET 10
